# Globetrotter — Decisions Log

## Milestone 0: Setup and Planning
- Destination chosen: Mauritius
- Why this destination: It is one of my dream vacation destinations and has many different activities to explore.
- Primary audience: Tourists and first-time visitors
- What this audience needs most: Clear ideas for events, attractions, and activities to build an itinerary quickly.
- 5-minute outcome goal: Visitors feel excited enough to book a trip and start planning right away.
- Key takeaway: Mauritius offers a variety of experiences, not just beaches.
- Desired feeling/tone: Awe and excitement
- One design decision that reflects the destination: A blue and green color palette inspired by the ocean and tropical scenery.
- Why this design fits: Mauritius is known for bright blue water and lush tropical landscapes.
- Wireframe format used (hand-drawn / Figma / other): Completed

## Milestone 1: HTML Structure
- Built 4 pages: Home, Attractions, Accommodations, and Gallery.
- Added consistent navigation and footer across all pages.
- Organized attractions/accommodations using repeatable card sections.

## Milestone 2: CSS Styling
- Applied a coastal color theme with consistent fonts, spacing, and card styling.
- Styled the hero banner, page sections, and gallery visuals.
- Added hover caption overlays for gallery photos.

## Milestone 3: Flexbox Layout
- Used Flexbox for nav, feature cards, attraction cards, accommodation list, and gallery grid.
- Set wrapping and spacing so layouts stay clean across different screen sizes.

## Milestone 4: Responsive Design
- Added media queries for tablet and mobile.
- Made nav adjust on smaller screens.
- Updated gallery layout to 3 columns (desktop), 2 (tablet), 1 (mobile).
