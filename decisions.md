# Globetrotter — Decisions Log

## Milestone 0: Setup and Planning
- Destination chosen: Mauritius
- Primary audience: Tourists
- One design decision that reflects the destination: The color scheme
- Wireframe format used (hand-drawn / Figma / other): completed

## Milestone 1: HTML Structure
- Built four pages: `index.html`, `attractions.html`, `accomodations.html`, and `gallery.html`.
- Used semantic sections (`nav`, `header`, `main`, `section`, `article`, `figure`, `footer`) to keep content organized.
- Kept a consistent navigation bar on every page for easy movement across the site.
- Structured attractions and accommodations as repeatable cards so the layout stays consistent.

## Milestone 2: CSS Styling
- Chose a coastal blue palette to match Mauritius beach and ocean vibes.
- Added card borders, rounded corners, and spacing to separate content clearly.
- Styled the home banner with overlaid heading text for a strong first impression.
- Kept styling in a single `styles.css` file to make updates easier across all pages.

## Milestone 3: Flexbox Layout
- Used Flexbox for the nav bar, homepage feature cards, attraction cards, accommodation list, and gallery grid.
- Set up wrapping behavior so layouts adapt naturally as screen size changes.
- Adjusted gaps and card sizing to improve readability and visual balance.

## Milestone 4: Responsive Design
- Added media queries for tablet and mobile breakpoints.
- Updated nav behavior on smaller screens (wrapping + centered logo row).
- Adjusted gallery card widths and image heights so images remain clear and not overly stretched.
- Ensured one-column gallery layout on small phones for better mobile viewing.

## Stretch Features
- Added hover caption overlays in the gallery for more interactivity.
- Expanded content depth by adding additional attractions and accommodation entries.
- Added more descriptive copy across page intros for a stronger travel-guide tone.
